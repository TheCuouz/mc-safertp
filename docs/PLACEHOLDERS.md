# PlaceholderAPI Placeholders

Requires PlaceholderAPI to be installed.

| Placeholder | Returns | Description |
|-------------|---------|-------------|
| `%safertp_cooldown%` | integer | Seconds remaining on cooldown (0 if ready) |
| `%safertp_can_use%` | `true`/`false` | Whether the player can use RTP right now |
